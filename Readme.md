## Game Content

缝合***Animal Party***和***Splatoon***：Blustoon = Animal Party + Splatoon ！

有2个玩家，1P2P，**游戏目标**是尽可能多的用喷枪染色！（喷就完事）

除了喷枪，还有武器锤子，两者都可以阻碍对方的行动喔~

共有三关，每关倒计时60s，时间结束地板会破碎，掉到下一关。
三关结束后触发结算界面~


> 游戏机制较多，可以慢慢发掘~~
